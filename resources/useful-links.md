# Useful Resources

## Overview

This section contains curated cybersecurity resources that support learning and understanding of key concepts, tools, and real-world threats.

The goal is to collect reliable and practical materials for continuous learning.

---

## 1. General Learning Platforms
- TryHackMe – ethical-hacking guided labs and learning paths https://tryhackme.com/ 
- Hack The Box – hands-on hacking labs and realistic environments https://www.hackthebox.com/ 

---

## 2. Web Security
- OWASP – industry-standard guidelines (e.g., OWASP Top 10) https://owasp.org/  
- PortSwigger Web Security Academy – free labs for web vulnerabilities https://portswigger.net/web-security  

---

## 3. Cybersecurity Fundamentals
- CISA – practical security guidance and alerts https://www.cisa.gov/  
- NIST – security frameworks and best practices https://www.nist.gov/cyberframework
- Security – benchmarks and CIS Controls https://workbench.cisecurity.org/
- No More Ransom – tools for ransomware recovery https://www.nomoreransom.org/pl/index.html

---

## 4. Threat Intelligence
- VirusTotal – scans files/URLs using multiple engines https://www.virustotal.com/ 
- abuse.ch – IOC feeds (malware, botnets) https://www.abuse.ch/
- Exploit Database – public database of exploits https://www.exploit-db.com/
- MITRE ATT&CK – attacker tactics and techniques https://attack.mitre.org/
- MITRE ATLAS – threats related to AI systems https://atlas.mitre.org/

---

## 5. Core Security Tools

### Network Scanning & Analysis
- Nmap – scans ports, services, and hosts https://nmap.org/  
- Wireshark – captures and analyzes network traffic https://www.wireshark.org/

---

### Penetration Testing / Red Team
- Metasploit – exploit development and testing https://www.metasploit.com/
- Burp Suite – intercepts and tests web traffic
- Kali Linux – OS with preinstalled security tools https://www.kali.org/
- Rapid7 – enterprise security solutions https://www.rapid7.com/

---

### Credential & Password Tools
- Mimikatz – extracts passwords from memory https://github.com/gentilkiwi/mimikatz
- Hashcat – GPU-based password cracking
- Hydra – login brute-force attacks

---

### OSINT / Reconnaissance
- Maltego – visualizes relationships between data https://app.maltego.com/
- theHarvester – gathers emails and domains
- Shodan – finds internet-connected devices

---

### Monitoring / SIEM / Detection
- Wazuh – log analysis and threat detection https://wazuh.com/
- Splunk – advanced log analytics
- Suricata – intrusion detection and prevention
- Zeek – behavioral traffic analysis

---

### Vulnerability Scanning
Nessus – scans systems for known vulnerabilities
OpenVAS – free alternative to Nessus

---

### Threat Intelligence Platforms
- MISP – shares threat data (IOC)
- OpenCTI – manages cyber threat intelligence

---

### Cryptography / Learning
- CrypTool 2 – learn encryption and crypto concepts https://www.cryptool.org/

---

### Payloads / Hardware Security
- PayloadStudio – generates payloads for USB attacks https://payloadstudio.hak5.org/community/
- Lab401 – tools for hardware hacking https://lab401.com/

---

### 6. IoT / OT Security Tools
- Shodan – discovers exposed IoT devices
- Censys – maps internet-exposed assets
- Nozomi Networks – industrial network monitoring
- Claroty – protects OT/ICS environments
