# Network Threats

## Overview

Network threats are malicious activities that target the confidentiality, integrity, or availability of data within a network.

Understanding these threats is essential for protecting systems and infrastructure.

---

## Common Network Threats

### Man-in-the-Middle (MitM)
- An attacker intercepts communication between two parties.

### Denial-of-Service (DoS / DDoS)
- Overloads a system or network to make it unavailable.

### Packet Sniffing
- Capturing network traffic to access sensitive information.

### Spoofing
- Impersonating a trusted device or user.

---

## Attack Vectors

- Unsecured Wi-Fi networks  
- Weak authentication  
- Misconfigured systems  

---

## Prevention Methods

- Use encryption (HTTPS, VPN)  
- Implement firewalls and intrusion detection systems  
- Secure network configurations  
- Monitor traffic  

---

## Key Takeaways

- Network threats target communication systems  
- Many attacks exploit weak configurations  
- Prevention requires layered security measures  

---

## Notes

Awareness of network threats helps in designing secure and resilient systems.
