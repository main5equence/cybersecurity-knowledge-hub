# Network Basics

## Overview

Network security relies on understanding how computer networks operate. A network is a system of interconnected devices that communicate with each other to share data and resources.

Understanding basic networking concepts is essential for identifying and preventing security threats.

---

## What is a Network?

A network connects devices such as computers, servers, and mobile devices to enable communication.

### Types of networks:
- LAN (Local Area Network)  
- WAN (Wide Area Network)  
- WLAN (Wireless Local Area Network)  

---

## Key Components

- **IP Address** – identifies a device on a network  
- **Router** – directs traffic between networks  
- **Switch** – connects devices within a network  
- **Firewall** – filters incoming and outgoing traffic  

---

## Data Transmission

Data is transmitted in small units called packets.

Each packet contains:
- source and destination address  
- data payload  

---

## Network Layers (OSI Model)

The OSI model divides communication into layers:

1. Physical  
2. Data Link  
3. Network  
4. Transport  
5. Session  
6. Presentation  
7. Application  

---

## Key Takeaways

- Networks enable communication between devices  
- Understanding components and structure is essential for security  
- Network layers help organize how data is transmitted  

---

## Notes

A strong understanding of network basics is crucial for analyzing vulnerabilities and securing systems.
