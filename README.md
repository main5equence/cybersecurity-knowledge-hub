# Cybersecurity Knowledge Hub

This repository is a space where I document and share knowledge related to cybersecurity.
It is designed to support the understanding of key security concepts through structured notes, presentations, and curated learning materials. The goal is to make complex topics more accessible and easier to understand, both for myself and others interested in cybersecurity.

---

## What you will find here

- Educational materials and presentations  
- Structured notes on cybersecurity topics  
- Explanations of key concepts and threats  
- Curated resources for further learning  
- Personal insights and reflections  

---

## Purpose

This project serves as both a personal learning portfolio and a knowledge-sharing space.
It reflects my ongoing development in cybersecurity and focuses on building a clear, practical understanding of how security concepts apply in real-world scenarios.

---

## Topics Covered

- Cybersecurity Fundamentals  
- Network and Infrastructure Security  
- Web Application Security (OWASP Top 10)  
- IoT & Operational Technology (OT) Security  
- Usable Security and Human Factors  
- Social Engineering and Insider Threats  
- Cyber Threats and Attack Vectors  
- Risk, Vulnerability, and Threat Analysis  
- Security Best Practices and Principles
